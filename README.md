Zenn is a lightweight music player built with HTML, CSS, and JavaScript.  
This project is created mainly to practice JavaScript and UI development, and to learn how to build small interactive components.

---

## Features (MVP)

- Play and pause the current song
- Repeat the current track
- Shuffle to play a random song
- Display the full song list
- Play a song by clicking on it
- Seek within the track using a progress bar
- Save playback state in localStorage

---

## Purpose

This project is built for:

- Practicing JavaScript fundamentals  
- Learning DOM manipulation  
- Understanding the HTML audio element and related APIs  
- Building simple UI components  
- Structuring a beginner-friendly JavaScript project  

Additional features may be added in the future.

## Technologies Used

- HTML5
